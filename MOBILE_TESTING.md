# Тестування мобільної версії

✅ **Оптимізація тексту:**
- Менші шрифти на мобільних

✅ **Мобільне меню:**
- Гамбургер меню на малих екранах
- Зручна навігація

## Як тестувати на телефоні:

### Варіант 1: Після деплою

1. Задеплойте на сервер (Railway + Vercel)
2. Відкрийте URL на телефоні
3. Увійдіть як продавець
4. Перевірте всі функції

### Варіант 2: Локально через мережу

1. **Знайдіть IP адресу вашого комп'ютера:**
   ```bash
   # macOS/Linux
   ifconfig | grep "inet "
   
   # Windows
   ipconfig
   ```

2. **Запустіть frontend з доступом з мережі:**
   Оновіть `frontend/vite.config.ts`:
   ```typescript
   server: {
     host: '0.0.0.0', // Дозволяє доступ з мережі
     port: 5173,
     // ...
   }
   ```

3. **На телефоні відкрийте:**
   ```
   http://ВАШ_IP:5173
   ```
   Наприклад: `http://192.168.1.100:5173`

4. **Для backend також налаштуйте CORS:**
   Оновіть `backend/src/index.ts`:
   ```typescript
   app.use(cors({
     origin: ['http://localhost:5173', 'http://ВАШ_IP:5173'],
     credentials: true
   }));
   ```

### Варіант 3: Chrome DevTools (для швидкого тесту)

1. Відкрийте сайт в Chrome
2. Натисніть F12 (DevTools)
3. Натисніть іконку мобільного пристрою (Ctrl+Shift+M)
4. Виберіть пристрій (iPhone, Samsung тощо)
5. Тестуйте інтерфейс

## Поради для продавців:

1. **Додайте на головний екран:**
   - iOS: Safari → Share → Add to Home Screen
   - Android: Chrome → Menu → Add to Home Screen

2. **Використовуйте в портретному режимі** (вертикально)

3. **Переконайтеся, що є інтернет** для роботи

## Проблеми та рішення:

**Проблема:** Сайт не відкривається на телефоні
- Перевірте, чи backend має правильний CORS
- Перевірте, чи frontend доступний з мережі

**Проблема:** Кнопки занадто малі
- Вже виправлено (мінімум 44px)
- Якщо все ще проблема, збільште в CSS

**Проблема:** Текст занадто дрібний
- Вже адаптивний
- Можна збільшити в `index.css` для мобільних

## Після деплою:

Після завантаження на сервер, сайт буде доступний з будь-якого пристрою з інтернетом. Просто відкрийте URL в браузері телефону.
