# Як перевірити сторінку продавця

## Крок 1: Створіть ларьок (якщо ще немає)

1. Увійдіть як адміністратор (admin / admin123)
2. Перейдіть на сторінку "Ларьки"
3. Натисніть "+ Додати ларьок"
4. Заповніть:
   - **Назва:** Наприклад, "Ларьок №1"
   - **Адреса:** Наприклад, "вул. Хрещатик, 1"
5. Натисніть "Зберегти"

## Крок 2: Створіть продавця

1. Перейдіть на сторінку "Продавці"
2. Натисніть "+ Додати продавця"
3. Заповніть форму:
   - **ПІБ:** Наприклад, "Іван Петренко"
   - **Логін:** Наприклад, "seller1"
   - **Пароль:** Наприклад, "seller123"
   - **Ларьок:** Виберіть створений ларьок
4. Натисніть "Зберегти"

## Крок 3: Додайте товари до ларька

1. Перейдіть на сторінку "Товари"
2. Натисніть "+ Додати товар"
3. Заповніть:
   - **Назва:** Наприклад, "Одноразка Vape 5000"
   - **Бренд:** Наприклад, "Vape"
   - **Тип:** Наприклад, "Одноразка"
   - **Ціна:** Наприклад, 250
   - **Кількість:** Наприклад, 10
   - **Ларьок:** Виберіть той самий ларьок, що й для продавця
4. Натисніть "Зберегти"

## Крок 4: Увійдіть як продавець

1. Вийдіть з системи (натисніть "Вийти")
2. На сторінці входу введіть:
   - **Логін:** `seller1` (або той, що ви створили)
   - **Пароль:** `seller123` (або той, що ви вказали)
3. Натисніть "Увійти"

## Що ви побачите:

- **Панель продавця** з картками:
  - Наявність товару
  - Моя виручка
  - Моя комісія (12%)
  - Продано сьогодні

- **Список товарів** з кнопкою "Продати" для кожного товару

- **Історія продажів** - ваші власні продажі

## Тестування продажу:

1. Знайдіть товар у списку
2. Натисніть кнопку "Продати"
3. Підтвердіть продаж
4. Перевірте:
   - Кількість товару зменшилась
   - Продаж з'явився в історії
   - Комісія оновилась

## Примітки:

- Продавець бачить тільки товари свого ларька
- Продавець бачить тільки свої продажі
- Комісія розраховується автоматично (12% від суми продажу)
